    Some hooks are only relevant for certain scenarios. To enable you to control the execution of a hook you can supply a tag value:
    
    \JAVA{\hspace*{1cm}\texttt{@Before("@SpecialTest")}}
    \CSHARP{\hspace*{1cm}\texttt{[BeforeScenario('SpecialTest')]}}
    \JAVASCRIPT{
        \hspace*{1cm}\texttt{Before(\LEFTCURLY tags: "@foo"\RIGHTCURLY, function () \LEFTCURLY} \\
        \hspace*{1cm}\texttt{\ \ // This hook will be executed before scenarios tagged with @foo} \\
        \hspace*{1cm}\texttt{\RIGHTCURLY)}
    }
    \RUBY{
        \hspace*{1cm}\texttt{Before("@foo") do} \\
        \hspace*{1cm}\texttt{\ \ \# This hook will be executed before scenarios tagged with @foo} \\
        \hspace*{1cm}\texttt{end}
    }
    
    Hooks tagged in this way will only run if the current scenario also has this tag on it.
